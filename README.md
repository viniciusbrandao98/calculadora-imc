# ⚖️ Calculadora de IMC

Um projeto prático desenvolvido para calcular o Índice de Massa Corporal (IMC) do usuário, fornecendo um diagnóstico rápido com base no peso e na altura inseridos.

## 🚀 Tecnologias Utilizadas

* **HTML5**: Estrutura de formulários e exibição de dados.
* **CSS3**: Estilização moderna, uso de Flexbox e design responsivo para dispositivos móveis.
* **JavaScript (ES6+)**: Lógica matemática para o cálculo do IMC e manipulação dinâmica do DOM para exibição dos resultados.

## 🛠️ Funcionalidades

* **Cálculo de IMC**: Processa os dados de peso (kg) e altura (cm) instantaneamente.
* **Classificação de Saúde**: Exibe se o usuário está no peso ideal, sobrepeso ou outras categorias de acordo com a tabela da OMS.
* **Interface Responsiva**: Layout que se adapta perfeitamente a qualquer tamanho de tela (celulares, tablets e desktops).
* **Interatividade**: Efeitos de hover e feedback visual ao interagir com os botões.

## 🧠 Aprendizados neste projeto

Este projeto foi fundamental para fortalecer conceitos de:
- **Lógica Matemática**: Aplicação da fórmula `peso / (altura * altura)`.
- **Condicionais em JS**: Uso de estruturas `if/else` para classificar os resultados.
- **Manipulação de Inputs**: Tratamento de dados numéricos vindos do usuário.
- **CSS Avançado**: Criação de uma interface limpa e intuitiva focada em UX.
